Architecture

Main Systems

WooCommerce Storefront

Products, stock, cart, orders, checkout, coupons, shipping, customer accounts, and product prices remain controlled by WooCommerce.

Theme Enhancements

The custom theme adds pack pricing, AJAX cart, low stock handling, rewards, account drawer, free delivery nudges, search, age checks, schema, and modern templates.

Android App

The app is native Android and uses live website data. It does not store WooCommerce product data internally.

App API Plugin

A private REST bridge handles login, rewards, orders, addresses, checkout transfer, app update checks, and app event tracking.

Business rules

Rules That Must Stay Consistent

Free Delivery

Free UK delivery unlocks at 14.95 GBP. Rewards should not reduce qualification.

kangoo_free_shipping_threshold()

Kangoo Rewards

1 point per 1 GBP earned. 100 points = 1 GBP. Minimum redemption is 100 points.

kangoo_rewards_points_per_pound_value()

Coupons

URL coupons use ?coupon=CODE. The reserved rewards code is kangoo-rewards.

kangoo_apply_url_coupon()

Low Stock

Show low stock messaging from 1 to 9 remaining and prevent selecting pack sizes above available stock.

kangoo_get_low_stock_message()

Pack Pricing

Pack tiers are ACF-driven. Cart pricing is applied by setting the unit price based on quantity tier.

kangoo_apply_pack_pricing_to_cart()

18+ Checks

The site has an age gate and checkout age verification. Order metadata stores the submitted verification.

kangoo_validate_checkout_age_verification()

Standalone plugin

Kangoo Helper Mascot

The mascot plugin is separate from the theme so it can be updated or disabled without changing storefront code. It appears as an animated pouch finder prompt, skips the Pouch Finder page, and remembers dismissal in localStorage.

  • Plugin: kangoo-helper-mascot/
  • Main file: kangoo-helper-mascot.php
  • Sprite: assets/kangoo-mascot-sprite.png
  • Storage key: kangoo_helper_mascot_dismissed
Core functions kangoo_helper_mascot_enqueue_assets() kangoo_helper_mascot_is_pouch_finder() kangoo_helper_mascot_render()

Standalone plugin

Kangoo App API

The API plugin is the bridge between the native app and WooCommerce. It keeps sensitive logic on WordPress and lets the app remain lightweight.

GET /config POST /event POST /login POST /register GET /me GET /rewards GET /orders POST /reorder GET/POST /addresses POST /checkout GET/POST /alerts POST /push-token
Checkout flow

The app posts cart items to the API. The API creates a short-lived transfer URL. The website rebuilds the WooCommerce cart and redirects to secure checkout.

Release

Safe Release Checklist

  1. Theme changes: upload changed PHP/CSS/JS files, clear cache, test homepage, shop, cart, checkout, account, and product pages.
  2. API plugin: package with forward slashes so WordPress sees kangoo-app-api/kangoo-app-api.php.
  3. APK: bump versionCode and versionName, build, rename to kangoo-pouches-vX.Y.Z.apk, upload to /app/android/.
  4. Updates: leave WooCommerce > Kangoo App APK URL blank to auto-detect the newest APK.
  5. Final checks: app update notice, app checkout transfer, app order source meta, free shipping nudge, rewards, coupons, and low-stock limits.
